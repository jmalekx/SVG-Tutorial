SVG Basics

Scalable Vector Graphics (SVG) is a markup language which is used to describe 2D graphics in XML format. They allow the creation of graphics which can be scaled to any size without losing their resolution/quality.


SVG can be scaled to any size without losing its resolution. This is because instead of being based on pixels, it is based on geometry (vector mathematics) An SVG will look clear on any screen resolution and also has a smaller file size than for example a standard image or GIF, using browser optimisations.

Syntax explanation:

To create an SVG, you simply need to use the <svg> HTML tag. An SVG can also be implemented using the <img> HTML tag, by importing and SVG file, for example <img src='file.svg'> When using SVG we must also define a coordinate system with the viewbox attribute for example <svg viewBox="0 0 100 100">, or we can simply just use the <width> and the <height> elements. A simple example of the full syntax of an SVG would be as follows: 


<svg width="100" height="100">
  <circle cx="50" cy="50" r="40" stroke="#B5C56D" stroke-width="3" fill="#E8A3B5" />
</svg>

Multimedia applications:

HTML5 Canvas

HTML5 Canvas is a graphical drawing API which aids in the creation of dynamic graphics and animations through the use of JavaScript. It is a drawing surface, although being just a HTML element. Canvas is simply, well, a blank canvas - a container in which you can put anything in, whether it be building a game in it, JavaScript graphics etc. It can also be styled in conujunction with CSS and JavaScript too.

Syntax explanation:

To create a HTML5 Canvas, you simply need to use the HTML <canvas> tag in your HTML markup. The height and width can be specified within this tag, although the proper way of doing this is specifying it within the javascript. For example: 

canvas.width=500;
canvas.height=300;

After setting this, we need to access the context if the canvas element; we can only draw using the getContext() variable. The ctx object is the object which we will use for this. For example: 

var canvas = document.getElementById("canvas");
var ctx = canvas.getContext("2d");


Drawing shapes, texts and images:
For the basics, we can create many shapes within this HTML5 Canvas, as the canvas is initially blank.

Here is an example code of how you would create a line, and the fuctions required for it to look as expected: 

ctx.beginPath();
ctx.moveTo(x1,y1);
ctx.fill();
ctx.stroke();
ctx.lineTo(x2,y2);


Essentially, the path is reset, a subpath with a given point is created, filled and stroked with certain fill style, and point added to subpath to connect and create a straight line.

Now to draw a rectangle: 

ctx.fillRect(x,y,width,height);
ctx.strokeRect(x,y,width,height);


We can style the rectangle's colour like so: 

ctx.fillStyle="pink";



To draw a circle we will use the following: 

ctx.beginPath;
ctx.arc(x,y,radius,startAngle,endAngle,counterclockwise);
ctx.stroke();


We can style the stroke colour like so: 

ctx.strokeStyle="pink";



For inserting text onto our canvas this is what we do: 

ctx.font="20px Arial";
ctx.fillText("text",x,y);



We can also create a text with stroke: 

ctx.font="20px Arial";
ctx.strokeText("text",x,y);

Multimedia applications:

Similarly to SVG, HTML5 canvas is widelt used in the creation of interactive and dynamic graphics. For example, drawing applications can easily be made through canvas, by tracing user mouse input throughout. Interactive user interfaces likewise can be created with sliders, buttons, etc and also implemented into data visualisations and infographics. Another thing worth noting is that canvas can also allow for development of 2D games and puzzles too, alongside VR (Virutal Reality) or AR (Augmented Reality) applications where 3D models can be utilised.
